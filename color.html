Benjamin Herb / Color

XYZ to CIELAB

κ = 903.3
ɛ = 0.008856

xr = X / Xwhitepoint
yr = Y / Ywhitepoint
zr = Z / Zwhitepoint

fx = xr1/3 * (xr > ɛ) + ((κ * xr + 16) / 116) * (xr <= ɛ)
fy = yr1/3 * (yr > ɛ) + ((κ * yr + 16) / 116) * (yr <= ɛ)
fz = zr1/3 * (zr > ɛ) + ((κ * zr + 16) / 116) * (zr <= ɛ)

L* = 116 * fy - 16
a* = 500 * (fx - fy)
b* = 200 * (fy - fz)
