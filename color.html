CIEXYZ to CIELAB

κ = 903.3
ɛ = 0.008856

xr = X / Xwhitepoint
yr = Y / Ywhitepoint
zr = Z / Zwhitepoint

fx = xr1/3 * (xr > ɛ) + ((κ * xr + 16) / 116) * (xr <= ɛ)
fy = yr1/3 * (yr > ɛ) + ((κ * yr + 16) / 116) * (yr <= ɛ)
fz = zr1/3 * (zr > ɛ) + ((κ * zr + 16) / 116) * (zr <= ɛ)

L* = 116 * fy - 16
a* = 500 * (fx - fy)
b* = 200 * (fy - fz)

CIELAB to CIEXYZ

δ = 6/29

fx = (L* + 16) / 116 + a* / 500
fy = (L* + 16) / 116
fz = (L* + 16) / 116 - b* / 200

X = Xwhitepoint * (fx3 * (fx > δ)) + (3*δ * (fx - 4/29) * (fx <= δ))
Y = Ywhitepoint * (fy3 * (fy > δ)) + (3*δ * (fy - 4/29) * (fy <= δ))
Z = Zwhitepoint * (fz3 * (fz > δ)) + (3*δ * (fz - 4/29) * (fz <= δ))
